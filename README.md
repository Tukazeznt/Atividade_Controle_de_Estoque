Este projeto foi desenvolvido com o objetivo de demonstrar, de maneira simples e didática, o funcionamento de arrays em JavaScript por meio de uma aplicação web básica.
A página foi construída utilizando HTML para a estrutura, CSS para a estilização visual e JavaScript para a lógica de programação, reunindo todas essas tecnologias em um 
único arquivo. A proposta principal é permitir que o usuário interaja com um array, podendo visualizar seu conteúdo, adicionar novos elementos e remover 
o último item inserido.

O array utilizado no projeto recebe o nome de frutas e é iniciado vazio. A partir da interação com os botões disponíveis na página, o usuário consegue manipular
esse array de forma dinâmica. Ao solicitar a visualização das frutas, o sistema verifica se o array está vazio e, nesse caso, informa essa condição ao usuário.
Quando o array contém elementos, todos os valores armazenados são exibidos na tela, juntamente com a quantidade total de itens presentes.

A adição de novos elementos é feita por meio de uma caixa de diálogo, na qual o usuário digita o nome de uma fruta. Esse valor é inserido no final do array, 
e a exibição é atualizada imediatamente para refletir a alteração realizada. Já a remoção de elementos acontece sempre a partir do último item do array.
Caso o usuário tente remover um elemento quando o array estiver vazio, o sistema exibe uma mensagem de alerta para evitar inconsistências.

A interface visual foi pensada para ser clara e organizada, com cores suaves e elementos centralizados, facilitando a compreensão e o uso da aplicação.
O projeto, além de apresentar o conceito de arrays, também auxilia no entendimento de funções, estruturas condicionais e manipulação do conteúdo da página através do JavaScript.
Dessa forma, a atividade se mostra adequada para fins educacionais, especialmente para estudantes que estão iniciando seus estudos em programação e desenvolvimento web.
